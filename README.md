# Android TV Applications Collection

Эта коллекция содержит популярные приложения для Android TV, оптимизированные для просмотра медиаконтента на больших экранах.

## Медиа плееры

### MX Player Pro v1.93.4 (armv7)
- **Размер**: 37.1 MB
- **Описание**: Профессиональный видеоплеер с поддержкой всех популярных форматов
- **Особенности**: Аппаратное ускорение, субтитры, многоязычная поддержка

### VLC Media Player v3.7.0 (armv7)
- **Размер**: 47.2 MB
- **Описание**: Универсальный медиаплеер с открытым исходным кодом
- **Особенности**: Поддержка всех кодеков, потоковое вещание, сетевые протоколы

### Vimu MediaPlayer v12.00
- **Размер**: 62.3 MB
- **Описание**: Оптимизированный медиаплеер для Android TV
- **Особенности**: Поддержка сетевых хранилищ, DLNA, удобное управление

## Установка

1. Убедитесь, что на вашем Android TV включена "Установка из неизвестных источников"
2. Скачайте нужные APK файлы на USB-накопитель или через облачное хранилище
3. Установите приложения через файловый менеджер
4. Наслаждайтесь просмотром контента!

## Советы

- Для лучшей производительности используйте MX Player Pro или VLC
- SmartTube Next рекомендуется для просмотра YouTube без рекламы
- TiviMate отлично подходит для IPTV провайдеров
- Используйте FileManager Plus для управления файлами на устройстве

## Требования

- Android TV 5.0+
- Минимум 2 ГБ оперативной памяти
- Стабильное интернет-соединение для стриминговых сервисов
